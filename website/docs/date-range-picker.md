# DateRangePicker - 选择日期段的日期选择器组件


* DateRangePicker

## 属性

属性名 | 类型 | 描述 | 是否必须 | 默认值 |  
------- | ------- | ------- | ------- | ------- | 
start | string | 时间, 如2017-5-1 | 否 | "" |
end | string | 时间, 如2020-2-11 | 否 | "" |
name | string |  | 否 | "" | 

## 事件
属性名 | 类型 | 描述 | 是否必须 | 默认值 | 参数说明 |  
------- | ------- | ------- | ------- | ------- | ------- |
onChange | func | 改变时间 | 否 | () => {} | 回调函数第一个参数为包含value, name, start, end的对象 |
